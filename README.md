## Description

Project for the course "Clean Code" at the school ESGI in fourth year.

This is the backend for a leitner system, a flashcard system that uses spaced repetition to help you remember things.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
